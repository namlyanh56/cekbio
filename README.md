# Panorama Cek Bio Bot

Bot Telegram untuk mengecek nomor WhatsApp secara massal. Mendeteksi:
- Nomor terdaftar/tidak
- Bio/status
- Tipe akun (Business / Regular)
- Meta Verified
- Official Business Account (OBA)

## Fitur

- 🔐 Login via Pairing Code (tanpa QR)
- 👤 Sender User: pakai bot WhatsApp pribadi (bisa multiple)
- 🌍 Sender Global: shared session milik owner
- 📊 Export hasil ke TXT dan Excel (XLSX)
- 📜 Riwayat pengecekan
- 📁 Dukungan file .txt untuk daftar nomor

## Prasyarat

- Node.js 18+
- NPM atau Yarn
- Telegram Bot Token (dari @BotFather)

## Instalasi

git clone https://github.com/your-repo/panorama-cek-bio.git
